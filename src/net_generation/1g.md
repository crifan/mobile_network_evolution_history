# 1G

* `1G`=`AMPS`/`TACS`/`NMT`/`C-Netz`
  * 信号编码方式：`FDMA`
    * `FDMA`=`Frequency Division Multiple Access`
  * 协议标准
    * `AMPS`
      * 美国
    * `TACS`
      * 英国
    * `NMT`
      * 北欧
    * `C-Netz`
      * 德国
        * ABC网络=A、B、C Network
          * A=A-line
          * B=B-grid
          * C=C-grid
        * 出现时间：1992年
