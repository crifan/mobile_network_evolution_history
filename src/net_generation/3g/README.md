# 3G

* 3G=UMTS/CDMA2000
  * 概述
    * 编码方式
      * 都是
        * CDMA
          * CDMA=Code Division Multiple Access
    * 2条技术路线 track
      * UMTS
        * 从GSM发展而来
      * CDMA2000
        * 从IS-95 和 D-AMPS 演化而来
          * IS-95
            * 别称：cdmaOne
        * 别称：
          * CDMA2000 1xRTT
          * IS-2000
  * 协议标准standard
    * `UMTS`
    * `CDMA2000`
