# 4G

* 4G=`LTE`/`WiMax`
  * `LTE`=`Long Term Evolution` (of mobile networks)
    * =`4G`=`the fourth generation`=`4G standard`
  * 最大速度：300 Mb/s
  * 2条技术路线
    * `LTE`
      * 演化自UMTS和CDMA2000
      * 目前主流是LTE
    * `WiMax`
      * WiMax=Worldwide Interoperability for Microwave Access
  * 协议标准
    * `LTE`
